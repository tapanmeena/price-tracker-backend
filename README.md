## Getting Started

### Install Dependencies
```bash
pnpm install
```

### Development
```bash
pnpm dev
```

### Build
```bash
pnpm build
```

### Production
```bash
pnpm start
```

## Environment Variables

Create a `.env` file in the root directory:

```env
PORT=3001
DB_HOST=localhost
DB_PORT=5432
DB_NAME=price_tracker
```
